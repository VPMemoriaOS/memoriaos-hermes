# Contributing

## Workflow

1. Update or reference the appropriate package in `memoria-spec`.
2. Implement the required functionality.
3. Validate against the package schema and test cases.
4. Submit changes.

## Rules

- Do not change the specification from this repository.
- Keep implementation deterministic where practical.
- Preserve provenance.
- Add tests for every new process.

## Commit Style

- feat:
- fix:
- docs:
- refactor:
- test:
- chore:
