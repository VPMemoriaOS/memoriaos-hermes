# MemoriaOS Hermes

> Reference implementation of the MemoriaOS specification using Hermes Agent.

## Purpose

This repository implements the MemoriaOS specification defined in the `memoria-spec` repository.

## Responsibilities

- Execute MemoriaOS processes
- Orchestrate transformation pipelines
- Generate canonical artifacts
- Validate outputs against specification packages

## Repository Layout

- `jobs/`
- `processes/`
- `prompts/`
- `templates/`
- `config/`
- `scripts/`
- `tests/`

## Related Repositories

- memoria-spec
- memoria-repository

## Development Principle

Implementation MUST follow the specification.
The specification never depends on this repository.
